Major Projects

StatusSpec

StatusSpec is a client plugin for Team Fortress 2 which helps to enhance the spectator experience by providing several different modules implementing functionality not available in the stock TF2 client. It has grown to be considered an essential tool for casting competitive matches, and is used by almost all of the top casting organizations like TeamFortressTV and eXtelevision and in coverage of some of the the competitive scene's largest events such as the ESEA-Invite Season 16 LAN finals, Insomnia52, and theGXL Universe 2014.

Website: http://github.com/fwdcp/StatusSpec

CompCtrl

CompCtrl is a server plugin for Team Fortress 2 that implements more streamlined competitive match management, offering features normally only seen in proprietary match mods used by leagues like CEVO and ESEA. It is currently in testing for adoption on ozfortress official match servers.

Website: http://github.com/fwdcp/CompCtrl

ExternalExtensions

ExternalExtensions is a client plugin for Team Fortress 2 currently in development with the goal of creating a WebSocket interface exposing internal game data to external applications. Currently planned functionality includes exposure of console I/O, player information, and game events.

Website: http://github.com/fwdcp/ExternalExtensions

Pressure

Pressure was a web API that helped to link subscriber status on Twitch channels to Steam accounts for the purposes of extending subscriber privileges within the game. It was used for providing special server privileges on the official server of popular TF2 streamer Ruwin until he retired from active play and the server was disbanded.
